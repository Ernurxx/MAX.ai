# Question diagram images

Place diagram images for practice test questions here. The app will show them when the question has an `imageUrl` set.

**Physics 2023 – suggested files:**

| File | Question | Description |
|------|----------|-------------|
| `ph10-circuit.png` | Q10 | Circuit with lamps 1–5 and switches K₁–K₄ (when K₂ and K₄ are closed, which lamps light up?) |
| `ph11-path.png` | Q11 | Path diagram: 0.1 km right, 60 m down, 4000 cm left, 20 m down (path and displacement) |
| `ph8-magnetic.png` | Q8 | Magnetic field and current-carrying conductor (force direction) |
| `ph21-dead-loop.png` | Q21–25 | «Мёртвая петля» (dead loop) diagram |

Copy your diagram screenshots or scans into this folder with the names above. The circuit diagram you provided can be saved as `ph10-circuit.png`.
